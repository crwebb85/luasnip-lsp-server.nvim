# luasnip-lsp-server.nvim

An LSP server to provide completion for luasnip snippets


https://github.com/user-attachments/assets/ce742aa7-4975-4ce4-ab58-70b6e0ae06db


## ⚡️ Requirements

-   Neovim >= **0.11.0**

## 📦 Installation

lazy.nvim

```lua
    {
        'crwebb85/luasnip-lsp-server.nvim',
        dependencies = {
            { 'L3MON4D3/LuaSnip' },
        },
        config = true,
    }
```
